# Algichar Zero Point - Wikipedia Style Website

একটি উইকিপিডিয়া স্টাইল ইনফরমেশনাল ওয়েবসাইট যা কিশোরগঞ্জ জেলার কুলিয়ারচর উপজেলার **আলগীচর জিরো পয়েন্ট** (৫ নং ওয়ার্ড, গোবরিয়া আব্দুল্লাহপুর ইউনিয়ন) এর তথ্য প্রদর্শনের জন্য তৈরি করা হয়েছে।

## বৈশিষ্ট্য (Features)
- 🏛 **Wikipedia UI Design:** উইকিপিডিয়ার মতো ক্লিন এবং প্রফেশনাল লেআউট।
- 📱 **Fully Responsive:** মোবাইল এবং পিসি সব ডিভাইসে সুন্দর দেখায়।
- 🔍 **SEO Optimized:** গুগল সার্চে (বাংলা ও ইংরেজি) সহজে খুঁজে পাওয়ার জন্য মেটা ট্যাগ যুক্ত।
- ⚡ **Lightweight:** শুধুমাত্র HTML এবং CSS ব্যবহার করে তৈরি, তাই খুব দ্রুত লোড হয়।

## প্রযুক্তি (Technologies Used)
- HTML5
- CSS3 (Custom Wikipedia Styling)
- Font Awesome 6 (Icons)
- Google Fonts (Noto Sans Bengali)
